# Carpeta de imágenes de productos

Aquí coloca las imágenes que descargues para usarlas en el proyecto.

Convenciones:
- Nombres en minúsculas, sin espacios (usa guiones o guion_bajo): `producto-azul.jpg`
- Optimiza el tamaño y formato (WebP/AVIF o JPEG/PNG comprimido).
- Referencia las imágenes desde el HTML/JS usando rutas relativas públicas, por ejemplo:
  - `public/images/products/mi-imagen.jpg` o desde el navegador `/images/products/mi-imagen.jpg` si sirves desde la carpeta `public`.

Ejemplo de uso en `src/js/index.js` (campo `image`):
```
image: "images/products/mi-imagen.jpg"
```
