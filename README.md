🛒 Tienda Online - Proyecto Web
📌 Descripción

Este proyecto consiste en una tienda online interactiva donde los usuarios pueden visualizar productos, agregarlos a un carrito de compras y realizar una solicitud de pedido mediante WhatsApp.

El sistema está diseñado para ofrecer una experiencia sencilla, intuitiva y funcional, enfocada en la interacción del usuario.

🚀 Funcionalidades principales
🛍️ Visualización de 40 productos
➕ Opción de agregar productos al carrito
📦 Cada producto incluye:
Imagen
Precio
Descripción
Botón de agregar al carrito
🧾 Carrito de compras dinámico
Contador de productos agregados
Visualización de productos seleccionados
Botón para finalizar compra
📲 Integración con WhatsApp
Al finalizar la compra, el usuario es redirigido automáticamente a WhatsApp con la solicitud del pedido
🧑‍💻 Manual de Usuario
🛒 Navegación en la tienda
El usuario puede explorar los productos disponibles en la página principal.
Cada producto cuenta con:
Imagen
Precio
Botón “Agregar al carrito”
➕ Agregar productos
Al hacer clic en Agregar al carrito, el producto se añade automáticamente.
El contador del carrito se actualiza en tiempo real.
🛍️ Carrito de compras
El usuario puede ver los productos agregados.
Se muestra el total de productos seleccionados.
Puede proceder a la compra mediante el botón “Finalizar compra”.
📝 Proceso de compra
Al hacer clic en Finalizar compra, el usuario es redirigido a un formulario.
Debe ingresar sus datos:
Nombre
Contacto
Información requerida
Al completar el formulario:
Se abre automáticamente WhatsApp
Se envía el mensaje con la solicitud del pedido
📞 Secciones adicionales
Contacto / Escríbenos: Permite comunicación directa con el cliente
Sobre nosotros: Información general del proyecto o negocio
⚙️ Mejoras y soluciones implementadas

Durante el desarrollo se solucionaron errores importantes:

🔧 Problema con el loader
El loader tenía un z-index demasiado alto, lo que impedía la interacción con los botones.
✔ Solución: Se ajustó el z-index para permitir la funcionalidad correcta.
🛒 Problemas en el carrito de compras
Se presentaban errores en la actualización del carrito.
✔ Solución: Se optimizó la lógica para:
Actualizar correctamente el contador
Mostrar productos correctamente

Desarrollar una tienda online funcional que permita:

Simular un proceso de compra real
Mejorar la experiencia del usuario
Integrar herramientas externas como WhatsApp
